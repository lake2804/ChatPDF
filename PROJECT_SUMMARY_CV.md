# ChatPDF - AI-Powered Document Chat Platform

## PROJECT SUMMARY FOR CV

**"CHATPDF – AI-POWERED DOCUMENT CHAT PLATFORM"**

**Description:** A full-stack intelligent document analysis platform built with FastAPI and React, enabling natural language conversations with documents through advanced RAG (Retrieval-Augmented Generation) technology.

**Tech Stack:**

**Frontend:** React 18, Vite, TypeScript (via JSDoc), Tailwind CSS, React Router, Lucide React

**Backend:** FastAPI (Python), LangChain, Google Gemini AI (LLM, Embeddings, Vision), Qdrant Vector Database, JWT Authentication

**AI/ML:** Google Gemini 2.0 Flash, text-embedding-004, Gemini Vision API

**Infrastructure:** Docker, Docker Compose, Qdrant Vector Database, RESTful API

**Key Features:**

**Document Intelligence:**
- Multi-format document support (PDF, DOCX, PPTX, TXT, Markdown, Images)
- Automatic text extraction and intelligent chunking
- Image OCR and captioning using Google Gemini Vision
- Smart document indexing with 768-dimensional vector embeddings
- Real-time document processing and indexing

**Advanced RAG Pipeline:**
- Semantic search with Qdrant vector database
- Context-aware answer generation using Google Gemini 2.0 Flash
- Source citation and reference tracking
- Multi-document query processing
- Customizable retrieval parameters (k-nearest neighbors)

**User Experience:**
- Secure authentication system (Login/Register)
- Beautiful, responsive UI with Tailwind CSS
- Real-time chat interface with message history
- Drag-and-drop file upload with progress indicators
- Document summarization with custom prompts
- Source display showing document references

**Smart Features:**
- Natural language question answering
- Document summarization
- Multi-document query support
- Real-time response streaming
- Error handling and user feedback

**Role:** Full-stack Developer – Developed complete frontend and backend architecture, including RAG pipeline implementation, vector database integration, authentication system, and responsive UI design.

**Deployment:** Local development with Docker Compose
